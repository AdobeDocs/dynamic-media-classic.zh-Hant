# Media Portal{#media-portal}

Adobe Medium門戶使公司能夠輕鬆地獲得，控制和向外部合作夥伴和渠道以及內部用戶分發經批准的資產。 基於瀏覽器的「自助」環境為門戶用戶提供管理員控制的對Adobe Dynamic Media Classic資產的「視圖」，以便以公司批准的格式輕鬆訪問上載、瀏覽、搜索、預覽和導出資產。

有關介質門戶（包括系統要求）的詳細資訊，請參閱 [使用Adobe Dynamic Media Classic媒體門戶](https://help.adobe.com/en_US/scene7/mediaportal/) <!-- (https://help.adobe.com/en_US/scene7/mediaportal/index.html) -->。

如需更多關於管理 Media Portal 的詳細資訊，請參閱[快速入門: Media Portal 管理](quick-start-media-portal-administration.md#quick_start_media_portal_administration)。
