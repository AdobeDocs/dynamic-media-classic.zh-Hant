# 導覽基本概念{#navigation-basics}

Scene7 Publishing System 畫面包含三個主要區域: 全域導覽列、資產庫，以及瀏覽面板/建置面板。

![導覽基本概念](/help/assets/gs_navigation_basics_popup_popup.png)

## 全域導覽列 {#global-navigation-bar}

全域導覽列，位於螢幕頂端，提供可執行各種不同工作的按鈕:

**選擇公司** 選擇您要從功能表存取的公司。如果您僅使用一家公司，這個按鈕便無法使用。

**上傳** 會開啓「上傳」畫面，讓您可以從電腦或網路上傳檔案至SPS。您可以從桌上型電腦或是透過 FTP 來上載檔案。請參閱上載檔案。

**從功能表中建立** 「建立」工作。會開啟「建置」面板，讓您建立資產類型。

**Publish** 開啓「Publish」(發佈)畫面，讓您可以將豐富媒體資產發佈至動態媒體影像伺服器以及其他動態媒體Classic SAAS基礎架構。請參閱發佈。

**工作** 開啓「工作」畫面，讓您檢查上傳和發佈工作的記錄。

**最近** 開啓「最近活動」畫面，讓您可以檢視最近修改資產的名稱以及最近上傳和發佈的工作。

**設定** 開啓「設定」畫面，讓您可以選擇設定或管理選項，以最佳化您使用Dynamic Media Classic的選項。請參閱設定。

**說明** 選擇說明選項：

**「說明&gt;說明」主題** 會開啓「說明」系統，其中含有關於您目前正在執行之工作的資訊。

**說明&gt;影片教學課程** 存取特定主題上的影片，以示範如何開始使用每個多媒體出版工作流程。

**說明&gt;說明內容** 開啓說明系統至「簡介」頁面。您可以點進「目錄」以尋找和檢視不同主題。

**說明&gt;關於** 您正在執行的Dynamic Media Classic版本。

**說明&gt;支援** 告訴您如何從Dynamic Media Classic取得技術支援。

**登出** 退出Scene Publishing System。

## 資產庫 {#asset-library}

使用「資產庫」來組織您所處理的資產。您可以建立檔案夾和子檔案夾來組織資產庫中的資產。

「資產庫」面板頂端的命令可用來尋找資產。您可以搜尋資產和篩選器來尋找資產。按一下「資產庫」底端的「垃圾桶」圖示 ，以檢查您所刪除的資產。

>[!NOTE]
>
>您可以按一下「資產庫」控制項，以開啟或關閉「資產庫」並取得更多螢幕工作空間。選取「展開/收合」控制項，以展開或收合面板。這些控制項位於面板的左側。

## 瀏覽面板/建置面板 {#browse-panel-build-panel}

「瀏覽/建置」面板是您進行大部分工作的地方。何者會出現在此面板中，端視您是在「瀏覽」模式或「建置」模式下工作而定:

**瀏覽模式** ：在「資產庫」中選取資料夾，以便在「瀏覽面板」中檢視並處理其內容。此面板提供了「檔案」、「預覽」、「全選」、「全部不選」和「排序」選單，可用來處理資產。您也可以移動滑桿或選取「格點檢視」、「清單檢視」 或「詳細檢視」 按鈕，以不同方式來檢視資產。選取「詳細檢視」按鈕，或是在詳細檢視中按兩下並開啟資產，即可從中執行資產相關檔案的作業。

**建立模式** 按一下「建立」，然後選擇一個選項，開始建立「建置」選單上的eCatalog、影像集、回轉集或其他數個可用項目之一。「建置」畫面開啟。您可以在進入「建置」模式之前或之後選取要使用的資產。
